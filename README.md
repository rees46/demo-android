# REES46 Demo Android

## Description

REES46 Demo Android - application to demonstrate working with SDK.

## Configure

Versions:
Java 22
Kotlin 2.0.0
Gradle 8.8
Android Gradle Plugin 8.5.1

Copy `google-services.json` file from [Firebase console](https://console.firebase.google.com/u/0/) to app module.
